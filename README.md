# RefineryAI: Document Intelligence Pipeline

A production-grade, multi-stage agentic pipeline for unstructured document extraction and intelligence. This system features classification-aware, spatially-indexed, provenance-preserving extraction, semantic chunking, and queryable knowledge for enterprise-scale document corpora.

## Description
RefineryAI ingests heterogeneous documents (PDFs, reports, images) and emits structured, queryable, spatially-indexed knowledge, supporting robust provenance and auditability for enterprise use cases.
